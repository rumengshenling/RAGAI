## 🐛 问题汇总

### 高优先级问题
1. **问题**: 智能问答回答质量不稳定
   - **原因**: 使用模拟大模型
   - **解决方案**: 接入真实本地大模型（ChatGLM3/Qwen）
   - **状态**: 待优化

### 中优先级问题
2. **问题**: 大文件导入时内存占用高
   - **原因**: 全量数据加载到内存
   - **解决方案**: 改为流式处理
   - **状态**: 已优化

### 低优先级问题
3. **问题**: Safari浏览器部分样式不兼容
   - **原因**: CSS前缀缺失
   - **解决方案**: 添加浏览器前缀
   - **状态**: 待修复
